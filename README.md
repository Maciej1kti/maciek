# Maciek

Mój system operacyjny. Prompty, kontekst, agenci.

## Struktura

```
├── context/        # Rodzina, firma, problemy
├── mentors/        # Filtry percepcji dla agentów
├── mems/           # Wspomnienia, prompty codzienne
├── ideas/          # Pomysły
├── plans/          # Plany
└── .claude/skills/ # Agenci
```
