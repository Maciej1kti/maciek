# Kontekst

Informacje które agenci mogą czytać żeby lepiej pomagać.

## Struktura

```
context/
├── rodzina.md      # Ola, Tomek, Julka
├── firma.md        # Biznes, projekty produkcyjne
├── problemy.md     # Aktualne wyzwania
└── README.md
```

## Jak używać

Agenci automatycznie czytają odpowiednie pliki gdy potrzebują kontekstu.
Aktualizuj gdy coś się zmienia.
