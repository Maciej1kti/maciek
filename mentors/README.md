# Mentorzy

Filtry percepcji. Agenci mogą patrzeć ich oczami.

## Użycie

Wywołaj mentora w kontekście:
- "Co powiedziałby Goggins?"
- "Jak Stoik by to widział?"
- "Maciek-gracz - jak zagrać tę sytuację?"
