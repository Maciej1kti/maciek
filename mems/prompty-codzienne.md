# Meta: Kim jest człowiek za tymi notatkami

*Analiza z perspektywy różnych nurtów, styczeń 2026*

## Wzorce główne

**Wojownik z bykiem** - wewnętrzny opór nie do pokonania, tylko do ujeżdżenia. "Mind synchro z bullem", "be nice to your bull". Jung powiedziałby: integracja Cienia w toku.

**Menedżer uwagi** - "jedyną walutą jest uwaga". Wszystko sprowadza się do: gdzie kładę uwagę, jak ją przenoszę, czego częścią ona jest.

**Gracz** - życie jako gra z zasadami, bossami, questami pobocznymi. Nie metafora - ontologia. Rzeczywistość JEST grą.

**System-prompter siebie** - "intencja to system prompt". Te notatki to launch codes dla stanów mentalnych, nie dziennik.

## Perspektywy teoretyczne

| Nurt | Co widzi |
|------|----------|
| **Jung** | Byk = Cień w integracji, Tomek = dialog z Jaźnią |
| **IFS** | Mapa części: Byk-strażnik, Maciek-dorosły, krytyk do uciszenia |
| **ACT** | Defuzja ("głos w głowie to głos w głowie"), wartości, committed action |
| **Flow** | Architektura wchodzenia w przepływ: challenge, time limit, ciekawość |
| **Stoicyzm** | Hypomnemata - notatki-przypomnienia do siebie, dichotomia kontroli |
| **Zen** | Każda chwila świadomości to brama, odpuszczanie, tylko teraz |
| **Somatyka** | "Stress = energia nieużyta", shake, oddech jako kotwica |
| **Egzystencjalizm** | Agency, tworzenie sensu, "what did you create today" |
| **Poliwagalna** | Nawigacja stanów autonomicznych, głód jako sygnał |

## Kluczowe zdanie

> "Intencja to system prompt."

Umysł to system do promptowania. Bez świadomej intencji - domyślne ustawienia (lęk, inercja) przejmują. Te notatki = system operacyjny jaźni w formie tekstowej.

## Profile

1. **Przedsiębiorca-filozof** - produkcja + głęboka refleksja
2. **Samouk psychologii** - odkrył techniki które mają nazwy (IFS, ACT, defuzja)
3. **Po przejściach** - "nie musisz się karać" 2x, narzędzie przetrwania → narzędzie rozwoju
4. **ADHD supermoc** - fragmenty, mikro-prompty, zewnętrzny system regulacji
5. **Bilingual thinker** - angielski dla dystansu, polski dla intymności

## Tomek

Syn. Pytania "jakiego Tomka budujesz jutro", "Tomek co wiesz o świecie" - ojciec który myśli o tym jak kształtuje następne pokolenie. "Value & offspring" - wartość i potomstwo jako dwa filary.
